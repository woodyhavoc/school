Below is a Factorial function example:

def fact(n)
  if n == 0
    1
  else
    n * fact(n-1)
  end
end

This function demonstrates the recursive abilities available with the Ruby language. As with any
factorial function, an original number is passed to the function. This number is multiplied by the
return value of the factorial function with the number - 1 passed as a parameter. This will continue
until the passed value is equal to 0, at which point the function will return a 1. Once this happens
all of the previous numbers will be mulitplied together, and eventually the final product will be returned.
