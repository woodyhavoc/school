Instructions for Windows:

1. Go to rubyinstaller.org.
2. Click "Download" at the top of the screen.
3. After the download is completed, run the installer and follow the on-screen instructions.

Instructions for Linux:

Execute one of the following commands based upon which distribution of Linux you are using:

Debian, Ubuntu, Linux Mint:

sudo apt-get install ruby

CentOS, Fedora, RHEL:

sudo yum install ruby

Gentoo:

sudo emerge dev-lang/ruby

Arch Linux:

sudo pacman -S ruby

Instructions for OS X:

Ruby comes pre-installed on OS X. It is possible that it is an older version though.
The recommended approach is to use HomeBrew to install a newer version of Ruby.
If you already have HomeBrew installed, continue to the install instructions, otherwise
here are the instructions to install HomeBrew:

HomeBrew Install Instructions:

Execute the following command from the terminal (note this will work with the pre-installed version of Ruby):

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

To install the latest version of Ruby, execute the following command from the terminal:

brew install ruby
